Testimonial Collector Web App

A simple web application that allows users to submit feedback and admins to view all submissions. Built using HTML, CSS, Bootstrap, and JavaScript (LocalStorage).



Features

User login → feedback form

Admin login → view all feedback

Collects name, email, phone, department, year, rating & message

Supports image upload and YouTube/Drive video links

All cards displayed in equal size

Admin can clear all feedback

Fully responsive pink-themed UI

How It Works

1. User/Admin logs in


2. User submits feedback (saved in LocalStorage)


3. Admin can view submissions in card layout


4. Admin can delete all feedback
